Web Page

Web page is a document available on world wide web .web pages are stored on webserver and can be viewed using a web browser. A web page contain different type of information including text, graphics, audio, video and hyperlinks .Hyper links are the link to other web pages.


There are two type of Webpages. They are:
  1. Static Web page

  2. Dynamic Web pag

Static Web page

User can only read the information but cannot change the information in the web page. Static webpages are created using only html. Static web pages are only used when the information is no more required to be modified.


Dynamic Web Page

Dynamic web page show different information at different point of time.ot is possible to change a portion of a web page without loading the entire web page.
